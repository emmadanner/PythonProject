# PythonProject

Need a simple way to track the tasks you need to do?  Use this task list script!
This script allows you to add and remove items from a list and mark them as completed!

--------------
Commands:

--help (or -h)
Shows how to use the script

--list
Displays the list of tasks with their completion status

--add "Task Name" "Task Description"
Adds an task to the list with the name and description you provide

--complete TaskNumber
Marks the selected task as completed

--delete TaskNumber
Deletes the selected task from the list
